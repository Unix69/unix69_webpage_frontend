Cos'è HTML?

HTML (HyperText Markup Language) è il linguaggio standard per creare pagine web. Utilizza tag per strutturare contenuti come titoli, paragrafi, immagini, collegamenti e tanto altro.

Struttura base di una pagina HTML

Una tipica pagina HTML è composta dai seguenti elementi:

  • <!DOCTYPE html>: indica il tipo di documento.
  • <html>: contenitore principale.
  • <head>: contiene meta-informazioni e collegamenti a risorse.
  • <body>: contiene i contenuti visibili della pagina.
<!DOCTYPE html>
<html lang="it">
  <head>
    <title>Esempio HTML</title>
  </head>
  <body>
    <h1>Ciao mondo!</h1>
  </body>
</html>

Esempi di tag HTML comuni

Tag Descrizione Esempio
<p> Paragrafo <p>Testo di esempio</p>
<a> Collegamento ipertestuale <a href="https://www.example.com">Visita</a>
<img> Immagine <img src="logo.png" alt="Logo">
<ul><li> Lista non ordinata <ul><li>Elemento</li></ul>
